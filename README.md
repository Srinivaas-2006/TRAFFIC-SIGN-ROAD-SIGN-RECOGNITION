# Traffic Sign Recognition - Initial Classification Model

## Project Description

This project focuses on developing an initial machine learning classification model for Traffic Sign Recognition.

The objective is to classify traffic sign images into their respective categories using image processing techniques and a machine learning algorithm.

The initial model is implemented using the Support Vector Machine (SVM) classification algorithm.

---

## Dataset

Dataset Used:

GTSRB (German Traffic Sign Recognition Benchmark)

The dataset contains real-world traffic sign images belonging to multiple classes.

Dataset Features:

- Different traffic sign categories
- Real captured images
- Multiple image samples per class
